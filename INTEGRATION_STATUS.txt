🆘 TROUBLESHOOTING:

Red Connection Dot?
  → Check: python main.py is running
  → Check: http://localhost:8000/api/ping in browser
  → Check: Firewall isn't blocking port 8000

Can't find flutter?
  → Run: flutter pub get
  → Run: flutter clean && flutter pub get

Port 8000 in use?
  → Run: netstat -ano | findstr :8000 (Windows)
  → Kill the process: taskkill /PID <PID> /F

💡 PRO TIPS:

✓ Use Generic Type Parameters
  final response = await apiService.get<MyType>(
    '/endpoint',
    fromJson: (json) => MyType.fromJson(json),
  );

✓ Always Handle ApiException
  try { ... } on ApiException catch (e) { ... }

✓ Use Dropdown Enums for Consistency
  final selected = RiskTolerance.fromString(userInput);

✓ Leverage DropdownHelper for Validation
  final income = DropdownHelper.castToDouble(userInput);

✓ Check Connection Before API Calls
  if (_backendConnected) { 
    // Make API call 
  }
